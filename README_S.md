### / **Task_01** /

##### _Задача 1:_
    За день машина проезжает n километров. Сколько дней нужно, чтобы проехать маршрут длиной m километров? 
    !!! При решении этой задачи нельзя пользоваться условной инструкцией if и циклами.

##### _Пример:_
>* Input:
>* - n = 700
>* - m = 750
>* Output:
>* - 2
